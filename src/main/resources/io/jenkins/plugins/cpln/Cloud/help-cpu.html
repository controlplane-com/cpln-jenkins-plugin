Specify the CPU resources in millicores (m) for the Jenkins agent container.

Minimum: 25 millicores (CPLN platform minimum)

Default: 200 millicores

Note: Do not exceed your CPLN organization's CPU quota. Check your quota in the Control Plane console under Org settings.

Examples:

  • 25m = minimum for basic workloads
  • 200m = good for light Jenkins agents
  • 500m = recommended for moderate build workloads
  • 1000m = 1 full CPU core for heavy builds
