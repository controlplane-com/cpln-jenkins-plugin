Specify the Memory resources in mebibytes (MiB) for the Jenkins agent container.

Minimum: 32 MiB (CPLN platform minimum)

Default: 512 MiB

Note: Do not exceed your CPLN organization's memory quota. Check your quota in the Control Plane console under Org settings.

Warning: Setting memory too low may cause OOM (Out of Memory) kills. The Jenkins inbound-agent typically needs at least 256 MiB for basic operation.

Examples:

  • 256 MiB = minimum recommended for Jenkins agents
  • 512 MiB = good for light to moderate builds
  • 1024 MiB = recommended for moderate to heavy builds
  • 2048 MiB = for memory-intensive builds (e.g., large Java projects)
